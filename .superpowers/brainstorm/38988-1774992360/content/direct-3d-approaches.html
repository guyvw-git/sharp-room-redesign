Direct 3D reasoning approaches

No image generation. The LLM reasons directly about the Gaussian splat and writes new Gaussians.

A

LLM Scene Understanding + Primitive Composition Recommended

Claude reads the PLY statistics (positions, colors, density distribution), reasons about the room's actual dimensions and layout, then composes the redesign from a library of properly-modeled 3D primitives — not boxes, but actual furniture geometry: cabinet frames with door panels, handles, toe kicks, drawer gaps, hinge positions.

The quality comes from correct 3D geometry + physically-based material appearance modeled directly as Gaussian attributes (specular layers, anisotropic grain, ambient occlusion in corners).

Pros

  • Fully direct — no 2D step anywhere
  • Room-aware: uses actual wall positions from PLY
  • LLM reasoning picks the right products/layout
  • Buildable now, iteratively improvable

Cons

  • Quality ceiling = procedural (no learned appearance)
  • Needs a rich 3D geometry library per object type
B

Text-Guided Gaussian Optimization (GaussianEditor-style)

Treat the Gaussian parameters as differentiable variables. Apply a text-conditioned loss — computed directly from Gaussian renders vs a target description — and gradient-descend the Gaussian positions, colors, scales, and rotations toward the target appearance. Pure 3D optimization, no 2D image generation.

Pros

  • Truly learned appearance, not procedural
  • Can produce photorealistic results
  • Operates directly on Gaussian parameters

Cons

  • Requires GPU + differentiable 3DGS renderer
  • Still needs a scoring model (CLIP, SDS loss) — those models are trained on images
  • Optimization can be unstable for large scene changes
C

3D Generative Model (text-to-scene 3DGS)

Use a model trained to generate large-scale 3DGS scenes directly from text — LucidDreamer, SceneWiz3D, or similar. These generate in Gaussian parameter space, conditioned on the text prompt and optionally the existing room geometry.

Pros

  • Full learned scene generation
  • Best possible quality ceiling

Cons

  • Scene-scale 3DGS generation is still research-stage
  • Hard to condition on an existing room structure
  • Large model requirements
