Three approaches to photorealistic room redesign

All use Gemini to generate reference imagery. The difference is how that becomes 3D Gaussian splats.

A

Gemini Image → TripoSR Mesh → Gaussian Sampling Recommended

For each piece of furniture in the redesign: Gemini generates a clean isolated product render → TripoSR (open-source image-to-mesh model) reconstructs a textured 3D mesh → surface Gaussians are sampled from the mesh with real UV colors baked in.

Pros

  • Real object geometry, not boxes
  • Colors come from photo-quality renders
  • Open-source, runs locally
  • Each object looks like the actual product

Cons

  • TripoSR needs to be installed (pip install)
  • Minutes per object on CPU
  • Mesh-to-Gaussian step loses some fidelity

Pipeline: Prompt → Claude reasons layout → Gemini renders each item → TripoSR → mesh → sample Gaussians → place in scene

B

Gemini Multi-View → Mini 3DGS Training

Gemini generates 8–16 views of each object from different angles → those images are used to train a mini 3DGS model per object (like the original NeRF/3DGS method) → the trained Gaussians are extracted and injected into the scene.

Pros

  • Highest possible quality
  • True photorealistic Gaussians
  • View-dependent appearance (SH)

Cons

  • Hours per object on CPU
  • Complex pipeline to orchestrate
  • Requires consistent multi-view generation from Gemini

Pipeline: Prompt → Claude reasons layout → Gemini renders 12 views per item → 3DGS training per item → inject

C

Gemini Full-Scene Render → Depth Lift

Instead of per-object generation, Gemini renders the entire redesigned room as a single high-quality image (with your actual room photo as reference) → a depth model (ZoeDepth/MiDaS) lifts it to 3D → Gaussians are placed at those depth positions.

Pros

  • Holistic, coherent scene composition
  • Gemini can see the actual room context
  • Fastest of the three

Cons

  • Depth estimation is noisy — Gaussians are imprecise
  • Single-view: no parallax, breaks on movement
  • Doesn't integrate cleanly with existing room structure

Pipeline: Prompt + room screenshot → Gemini redesigns full scene → ZoeDepth → Gaussian point cloud → inject
